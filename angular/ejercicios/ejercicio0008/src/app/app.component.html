Integración con BootStrap 3 y JQuery

Instrucciones

  1. Crear la app: ng new ejercicio0008 --skip-install --skip-git
  2. cd ejercicio0008
  3. Instalar paquetes:yarn add bootstrap3@ jquery angular2-uuid
  4. Comprobar la instalación:yarn
  5. Arrancar la app:ng serve -o
  6. Parar la app: CTRL+C
  7. Crear una versión de producción:ng build -prod
  8. Ir al directorio de distribución:cd dist
  9. Publicar la app en Internet:surge
